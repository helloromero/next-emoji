## next-emoji

a *very* simple example of [sockets.io](https://socket.io/) running inside a [next.js](https://nextjs.org/) project.


## how to run

`npm i` then `npm run dev`

## how to view

open [http://localhost:3000](http://localhost:3000) with your browser.
